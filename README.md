# 🏋️‍♂️ Fitness Pro – Health & Fitness Web App

A responsive fitness and wellness web application featuring:
- 🧮 BMI Calculator
- 🥗 Indian Food Calorie Calculator
- 💪 Prebuilt Weekly Workout Plans
- 🌿 Health and Wellness Tips
